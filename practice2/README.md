# 🛡️ КиберБезопасность — SPA (Лабораторная работа №1)

**Предметная область:** Безопасность в сети Интернет  
**Стек:** React, react-router-dom, axios, json-server

## Описание

SPA-приложение — каталог угроз информационной безопасности.  
Реализован полный CRUD: просмотр, добавление, редактирование, удаление угроз через RESTful API.

## Сущность Threat (Угроза)

| Поле | Тип | Описание |
|---|---|---|
| id | number | Уникальный идентификатор |
| name | string | Название угрозы |
| category | string | Категория атаки |
| severity | string | Уровень угрозы (Критическая / Высокая / Средняя / Низкая) |
| year | number | Год актуальности |
| description | string | Описание механизма атаки |
| prevention | string | Меры защиты |

## Маршруты SPA

| URL | Компонент | HTTP-метод |
|---|---|---|
| `/` | Home.js | GET, DELETE |
| `/detail/:id` | Detail.js | GET |
| `/add` | Form.js | POST |
| `/edit/:id` | Form.js | PUT |

## API-эндпоинты

| Метод | URL | Действие |
|---|---|---|
| GET | /items | Получить все угрозы |
| GET | /items/:id | Получить угрозу по id |
| POST | /items | Создать новую угрозу |
| PUT | /items/:id | Обновить угрозу |
| DELETE | /items/:id | Удалить угрозу |
